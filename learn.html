Enhance your QA knowledge in just 11 Days

1. Manual Testing ⏱ 7 Days  [1.5–2 hrs daily]

In this course, you will learn the fundamentals of software testing. It is recommended that you take detailed notes as you progress through the material, as these notes will be useful when you begin working as a QA engineer.

Sample CV Template
2. API Testing ⏱ 1 Day  [Approx. 2 Hours]

This course will provide a comprehensive understanding of APIs and how to test them using the POSTMAN tool. By the end of this course, you should have a solid foundation in API testing and be able to effectively utilise the POSTMAN tool.

3. Performance Testing ⏱ 1 Day  [Approx. 2 Hours]

In this course, you will get a good understanding of Performance Testing and the JMeter tool for Load and Performance Testing.

📌 Watch the JMeter 2021 Masterclass video until "1st JMeter Test" — i.e. up to the 28:57 mark (Video 2).
4. Test Automation ⏱ 1 Day  [Approx. 2 Hours]

Learn test automation using the Cypress framework and write your first automated test. Before starting, make sure you have Node.js, VS Code, and basic JavaScript knowledge.

📌 Prerequisite: Watch the JavaScript tutorial below before starting the Cypress course.
5. Scrum ⏱ 1 Day  [Less than an hour]

In this course you will get a better idea of Agile (specifically Scrum) methodology for Software Development.

🎁 Bonus — Free Certification on Cypress

After completing the courses above, you can earn a free Cypress certification from Test Automation University. Complete the Introduction to Cypress course and receive a certificate you can add to your CV and LinkedIn.
